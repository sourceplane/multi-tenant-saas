post-merge main CI run `26387697533` did not include `db-migrate` jobs. The
downloaded plan artifact contained only:

- `db.*.verify`
- `db-tests.dev.verify`
- `contracts.*.verify`
- `contracts-tests.dev.verify`

Local reproduction from main:

```bash
/Users/irinelinson/.local/bin/kiox -- orun plan --changed \
  --files packages/db/src/migrations/040_projects_core/up.sql \
  --trigger github-push-main \
  --intent intent.yaml
```

By contrast:


`Migration Apply` step. This suggests the current `spec.paths:
packages/db/src/migrations/**` entry is not participating in Orun changed
component selection.
