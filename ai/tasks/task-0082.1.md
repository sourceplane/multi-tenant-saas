## Objective

`apps/web-console-next` to actually produce

## Architect Brief

- Product bar: **Cloudflare Workers / Pages delivery quality** — the build
  must emit exactly what the established `cloudflare-pages-turbo` component
  contract consumes, with no bespoke deploy code.
- User moment improved: every Task 0082 surface (Cmd-K, designed
  `PreconditionInsight`, Zod create flows) actually reaches a Pages URL on
  dev/stage/prod, not just on a local `next build`.
- You have full latitude on: pinning the `@opennextjs/cloudflare` version,
  ordering build steps inside `package.json scripts.build`, whether to use
  the adapter's `npx opennextjs-cloudflare build` or a CLI wrapper, and any
  `turbo.json` outputs entry needed to make the orun
  `verify-build-output` step recognize the produced assets.
- You **must not**: change `component.yaml`'s `type`, `outputDir`,
  `projectName`, `environmentBuildVar`, `smokeCommand`, or any field other
  than (optionally) `installCommand` if the adapter requires an additional
  install hook; touch any other component, worker, contract, package, app,
  Terraform, policy, db, or api-edge; alter `apps/web-console`; remove or
  alter the verifier's prerender fix on `apps/web-console-next/src/app/layout.tsx`.
- Failure modes that invalidate this PR even if build passes:
  - Diff escapes `apps/web-console-next/**`, root `turbo.json`,
    `pnpm-lock.yaml`, and `ai/**`.
  - `apps/web-console` byte-changed.
  - `smokeCommand` semantics change (must keep `'Sourceplane Console'`
    string in the rendered HTML response of `/`).
  - Verify-deploy job still red on dev/stage/prod.
- Recommended pattern: add `@opennextjs/cloudflare` (latest stable
  compatible with Next 15.0.3), set `scripts.build` to
  `next build && opennextjs-cloudflare build`, and either (i) add a
  per-package `turbo.json` under `apps/web-console-next/` that overrides
  `outputs` to `[".open-next/**", ".next/**"]`, or (ii) extend the root
  `turbo.json` `tasks.build.outputs` to include `.open-next/**`. Prefer
  (i) to avoid widening every package's build cache surface.
